## 使用教程

以下是 AUT0RICH 的基本使用教程，仅供学习和研究参考。**请确保在使用前仔细阅读“使用须知”，并严格遵守相关法律法规和平台规则**。

### 1.配置OCR环境
- 运行setup_python_env.bat
- 运行install_packages.bat
### 2.安装并运行程序
- 运行AutoRichUv2.exe安装程序
- **以管理员身份**运行安装完的程序
  - 运行run_ocr_service.bat(每次使用前都需要运行来激活OCR识别程序)
- 开始使用
  - 开始使用前确保屏幕分辨率缩放为100%
